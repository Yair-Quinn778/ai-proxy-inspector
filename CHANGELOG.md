## v1.0.0

**Initial Release**

- 🚀 Full-scope AI node detection engine
- 🔍 **IP Information** — IPv4, ASN, ISP, geolocation, coordinates, timezone
- 🌐 **DNS Check** — DNS server detection and leak analysis
- 📡 **WebRTC Check** — Browser-side local IP leak detection
- 🤖 **AI Services** — ChatGPT, Claude, Gemini, Grok, Perplexity, Copilot, Poe, DeepSeek availability testing
- 🛡️ **Risk Analysis** — Google risk, Cloudflare risk, IP purity, blacklist detection
- 🎬 **Streaming** — Netflix, Disney+, Hulu, HBO Max, Prime Video, Apple TV+, Spotify, YouTube Premium access testing
- ⚡ **Speed Test** — Ping, download/upload speed, jitter, packet loss measurement
- 📊 **Scoring Engine** — Weighted scoring system (S/A/B/C/D/F grades)
- 🌍 **i18n** — Full Chinese (zh-CN) and English (en-US) support with runtime switching
- 💾 **Local SQLite** storage for detection history
- 🎨 **Dark theme** UI with glass morphism effects
- 📦 **Windows** MSI and NSIS installers
- 🤖 **GitHub Actions** CI/CD pipeline
